## 🌍 Problem Statement

Carbon emissions from daily activities — travel, food, energy, and shopping — are the leading drivers of climate change. Most people have no idea how much CO₂ their lifestyle produces. **CarbonTrace** makes that invisible footprint visible, comparable, and actionable.

---

## ✨ Features

- **6-input Carbon Calculator** — car travel, flights, electricity, diet, shopping, public transport
- **Live Visual Gauge** — animated SVG arc showing your emission level at a glance
- **Category Breakdown** — color-coded bars for each emission source
- **Personalised Tips** — recommendations based on your top 2 highest emission categories
- **Global Comparison** — see how you stack up against India, Global, EU, and USA averages
- **Pledge System** — commit to one climate action per month
- **Fully Responsive** — works on mobile, tablet, and desktop
- **Accessible** — ARIA labels, keyboard navigation, reduced motion support

## 🏃 Getting Started

### Run locally

```bash
# Clone the repo
git clone https://github.com/your-username/carbontrace.git

# Open in browser
cd carbontrace
open index.html
```

### Deploy on GitHub Pages

1. Push code to a public GitHub repository
2. Go to **Settings → Pages**
3. Set source to **Deploy from branch → main → / (root)**
4. Your app will be live at `https://your-username.github.io/carbontrace`
